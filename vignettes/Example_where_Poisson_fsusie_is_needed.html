These data are based on sample real GTeX RNAseq counts where we model the effect of a SNP on these count using Binomial thinning.

Those are simulated data of structured counts \(Y\) each row being the observed “profile” for an individual and in \(X\) we store the genotype data, among those the effect SNP is
